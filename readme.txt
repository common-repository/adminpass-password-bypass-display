AdminPass allows administrators to bypass password protection and admins/editors to view passwords for protected pages, posts, and custom post types.

== Description ==

AdminPass, developed by Belov Digital Agency, is a user-friendly plugin that simplifies password management for WordPress pages, posts, and custom post types. It streamlines the user experience for administrators and editors, granting access to powerful features:

1. **Bypass password protection**: Administrators can effortlessly access password-protected content without entering a password, ensuring seamless content management.
2. **Display password**: Administrators and editors can view the password for protected pages, posts, and custom post types directly in the editor, saving valuable time.

AdminPass is the ultimate solution for WordPress professionals looking to enhance their website management capabilities. Trust Belov Digital Agency to deliver innovative tools that boost productivity and streamline workflows.

== Installation ==

1. Upload the `adminpass` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The plugin will automatically provide the features without any additional settings.

== Frequently Asked Questions ==

= Is AdminPass compatible with custom post types? =

Yes, AdminPass works with custom post types in addition to pages and posts.

Yes, AdminPass can be used on a multisite installation.
